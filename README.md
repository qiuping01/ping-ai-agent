# 注意

+ 如果你本地安装了 steam++（Watt Toolkit）需要卸载，会占用端口。只保留 dev-sidecar 就可以了。
+ 该加速器如果在电脑关机的时候没有退出，会导致再开机后无法上网，只要再打开该加速器就好了。
